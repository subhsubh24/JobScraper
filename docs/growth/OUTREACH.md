# Strategic outreach — curated, human-reviewed email drafts (the Growth Agent loop)

A high-leverage channel the Growth Agent MAY run: a FEW 1:1, deeply-personalized outreach
emails to genuinely strategic targets, drafted as Gmail DRAFTS for the OWNER to review and
send. NOT cold-email at scale.

## Hard rails (non-negotiable)
1. **DRAFT ONLY** — the human sends; the agent never sends (the Gmail tool is
   `create_draft`); no auto-send ever.
2. **HIGH-CONFIDENCE + STRATEGIC only** — draft only if you can name the SPECIFIC real
   recipient + why THEY'd care + the realistic reply you anticipate; if you can't name all
   three, don't draft.
3. **A FEW per run MAX** — never a blast, never a bought/scraped list, never the same target
   twice without a real new reason; some runs = zero drafts (correct).
4. **REAL, publicly-appropriate contacts only** — use WebSearch to find the target + their
   PUBLISHED professional contact; NEVER invent/guess/scrape a personal email or harvest
   PII; if only a contact form exists, draft for the owner to paste.
5. **HONEST + COMPLIANT** — every claim TRUE (no invented metrics/traction/social proof),
   identify who you are + clear ask + easy opt-out (CAN-SPAM/GDPR-clean), on-brand voice.
6. **Pre-launch** every link points to the PUBLIC waitlist/"coming soon", never the
   gated/unfinished app.
7. **maker≠checker** — run each draft through the independent reviewer (on-brand? honest?
   genuinely strategic vs spam? compliant?) before queuing.

## Target types (examples — adapt to Career Operator; NOT a quota)
Genuinely strategic, with a real reason to care:
- **Journalists / publications** that actually cover careers, job-search, the labor market,
  HR-tech, or AI-for-work.
- **Integration / distribution partners**: ATS / job-board APIs (Greenhouse, Lever, Ashby),
  university career centers, coding bootcamps, outplacement / career-coaching firms.
- **Creators / communities** whose audience overlaps: career & tech-career creators,
  job-search YouTubers/TikTokers, interview-prep educators, relevant subreddit/Discord mods
  (engage authentically, never spam).
- **Newsletter curators** in careers / tech-careers / job-search.

NOT random businesses, generic "founders," scraped consumer emails, or anyone with no real
reason to care.

## Draft format (what the owner receives)
One Gmail DRAFT per target containing —
- **Target:** name / org / role + PUBLIC contact (or "contact form: <url>")
- **Why this target now** + the anticipated reply
- **Confidence:** high only
- **The Email:** subject + personalized body + ask + opt-out

Track in the `GROWTH_STATUS` outreach block (real numbers; replies owner-reported, never
fabricated); durable lessons in `GROWTH_MEMORY.md`.

## When NOT to
No genuinely strategic, researchable target this run → do NONE. A quiet run with zero
outreach drafts is success; a pile of generic cold drafts is failure.
